# 如何登录到 Vyatta

可使用 Web GUI 或直接通过 SSH 登录来访问 Brocade NFV（网络功能虚拟化）。

## 向 SoftLayer 专用网络进行认证

1. 在浏览器中输入**详细信息**屏幕中的专用 IP 地址（**网络 > 网关设备**菜单），使用 https:// 作为前缀（例如，https://10.54.207.131/）。请注意，您可能会收到证书错误，因为设备不使用公用证书。接受错误以转至 Web 页面。

2. 在 Web 门户网站中，针对 **vyatta** 用户输入**设备**菜单下**密码**选项卡中的凭证。您将登录到 Brocade 仪表板。
